# Secure Cloud Platform

## Goal
A cloud-ready, microservices-based platform demonstrating:
- Security
- Cryptography
- Cloud-native deployment
- CI/CD automation

## Tech Stack
- Frontend: React
- API Services: ASP.NET Core
- Cryptography Service: Python
- Containerization: Docker
- Orchestration: Docker Compose
- CI/CD: GitHub Actions
- Target Cloud: AWS (future)

## Deployment Model
Local Docker-based deployment simulating a single cloud VM.
